小米平板刷机指南

# 一、常规刷机步骤

# 1、刷入第三方Recovery - TWRP

步骤一、解除BL锁

  1. 打开「开发者模式」。 进入设置，点击我的设备 -> 点击全部参数 -> 一直点版本号，直到出现处于开发者模式
  2. 打开「USB调试」和「OEM解锁」。进入「设置 - 更多设置 - 开发者模式」。OEM解锁必须打开不然没办法刷入第三方Recovery
  3. 下载TWRP工具包。这里我下载的是
  4. 刷入第三方Recovery
# 重启进入 FastBoot 模式
> adb reboot bootloader

# 解锁 flash （这一步一定要确保OEM解锁打开）
> fastboot flashing unlock

# 检查是否解锁成功。出现以下输出表示BL解锁成功
> fastboot oem device-info
(bootloader) Verity mode: true
(bootloader) Device unlocked: true
(bootloader) Device critical unlocked: false
(bootloader) Charger screen enabled: false
(bootloader) Console enabled: false

# 开始刷入twrp
> fastboot flash recovery twrp.img

# 重启设备并进入Recovery
> fastboot reboot recovery

# 二、MIPAD1刷机指南

1）刷入TWRP

# 进入FastBoot，刷入twrp
> fastboot flash recovery twrp-3.4.0-0-mocha.img

2）四清

进入TWRP，点击Wipe -> Advance Wipe，选中前四个，滑动开始清理旧系统

3）刷入分区包

四清完成后，回到首页，点击install，选择前面已拷贝的文件 mocha_repartition_1_2GB_RahulTheVirus.zip，并滑动刷入，刷入完成后，会自动重启

4）刷入tos.img

重启进入FastBoot，刷入tos.img

# 进入FastBoot，刷入twrp
> fastboot.exe flash tos tos.img

# LineageOS 16.0 刷机步骤

LineageOS 16.0

Requirements:

  1. active system1
  2. merged system partition
  3. BRAIN, READING SKILL and presence of basic knowledge about firmware (in the absence need learn to independently)

Instruction

  1. download TosForPSCI-0.1.zip (opens new window)
  2. unzip TosForPSCI-0.1.zip
  3. download recovery-03.11.2018 (opens new window)
  4. go to fastboot mode
  5. execute command: fastboot flash tos your path to tos.img
  6. execute command: fastboot flash recovery your path to recovery-03.11.2018.img
  7. reboot to new recovery(u can execute fastboot reboot holding vol+ button)
  8. if you are on previous LineageOS 16.0 build then you may install the ROM without wipes. If you're not, make full wipes then
  9. install current LineageOS 16.0 build
  10. reboot to the system
