# 1. win10 和 offic 激活指南

通过自己搭建『KMS服务』，作为 win10 和 office 的注册服务器，可持续激活

# 1.1 搭建KMS服务

# 拉取kms服务镜像
$ docker pull mogeko/vlmcsd
# 启动kms服务
$ docker run -d -p 1688:1688 --restart=always --name vlmcsd mogeko/vlmcsd

# 1.2 激活win10

# 设置kms服务，默认1688端口
> slmgr /skms 192.168.31.17
# 设置安装密钥，更多密钥：https://docs.microsoft.com/zh-cn/windows-server/get-started/kmsclientkeys
> slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
# 立即激活
> slmgr /ato
