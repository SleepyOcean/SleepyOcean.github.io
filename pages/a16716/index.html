# 2）系统配置

1、关闭自动更新：

步骤一：以管理员身份运行 Windows PowerShell；

步骤二：执行sc.exe config wuauserv start= disabled，禁止 Windows 启动更新服务；

步骤三：执行sc.exe stop wuauserv，停止当前正在运行的 Windows 更新服务。

步骤完成，你以后再也不会见到 Windows 10 自动更新了。若你想验证一下，可以前往 Windows 10 的更新界面，此刻会看到提示「遇到错误：更新服务无法正常运行」。

**适用对象：**本方法仅适用于终止未来可能有的更新，无法阻止已进入待安装状态的更新。若你的系统已有更新进入安装状态，请先安装完成该更新后再使用以上步骤。

**恢复自动更新：**若你想恢复自动更新，只需输入 sc.exe config wuauserv start= auto 即可。
